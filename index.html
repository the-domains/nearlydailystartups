Nearly Daily Startups

様々なWebサイト上に羅列されているデータを簡単な操作で成形し、スプレッドシート化することができるサービスだ。

いくつかの機能がついているが、特にMagicは素晴らしい。Webからログインしなくても使えるこのMagicを利用すると、入力先のサイトの情報をひとまとめにして表を作成してくれる。

Nearly Daily Startupsについて

このサイトでは（できれば）ほぼ毎日、スタートアップのサービスや製品、その他トレンドについての情報を紹介します。

This is a website to introduce a lot of startups, but (hopefully) one post per nearly daily. In the end of the day, we hope to have a directory with tons of startups!
